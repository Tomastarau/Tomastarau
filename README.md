## Some projects i did

### FaceForge

A proof of concept that turns a face photo into Dragon's Dogma 2 character creator sliders.

It started as a fun technical experiment around facial landmarks, proportions, and feature mapping. I shared it publicly, got real user feedback, and iterated on validation, stability, and privacy handling.

Repo: [FaceForge](https://github.com/Tomastarau/FaceForge)

### UFC Data Analytics

A data project built around UFC fights, fighters, and performance analysis.

The goal is to explore rankings, comparisons, and Elo-based prediction logic through a dashboard-driven approach.

Repo: [UFC-Data-Analytics](https://github.com/Tomastarau/UFC-Data-Analytics)

### Resume Matcher experiments

A more flexible AI-assisted version of Resume Matcher.

My main contribution was adding a persistent master profile system: instead of rewriting a resume from scratch every time, the tool can store experience and projects, select the most relevant ones for a job post, and reformulate them more clearly without inventing anything.

Repo: [Resume-Matcher](https://github.com/Tomastarau/Resume-Matcher)
